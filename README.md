
Proxima Reverie
============
A tactical RPG engine similar to X-Com or Final Fantasy Tactics

TECHNOLOGY
==========
Proxima Reverie primarily runs in the browser, using an HTML5 canvas element driven by Javascript.  Supporting technologies TBD.
